# Draw.io Standards

Draw.io / diagrams.net is the primary tool for this repository.

## Usage

Use Draw.io for:
- primary architecture diagrams
- editable diagram deliverables
- imported Mermaid skeletons
- reusable library validation
- PNG/PDF/SVG exports

## Mermaid Import

In Draw.io:

```text
Arrange → Insert → Advanced → Mermaid
```

Mermaid provides structure. Draw.io provides visual refinement.

## File Handling

Store:
- Mermaid source in `diagrams/mermaid`
- Draw.io source in `diagrams/drawio`
- Visio files only when needed in `diagrams/visio`
- exports in `diagrams/exports`
