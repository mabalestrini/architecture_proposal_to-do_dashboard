# Stencil Catalog

## Stencil Families

| Family | Prefix | Purpose |
|---|---|---|
| Areas | AREA_ | Major operational domains |
| Zones | ZONE_ | Network/security segments |
| Actors | ACTOR_ | Customers, workers, partners, admins, guests |
| Security Controls | SEC_ | Firewalls, WAF, Prisma, IDS/IPS |
| Network Infrastructure | NET_ | Routers, switches, ACI spine/leaf, load balancers |
| Connectivity | CONN_ | MPLS, Meraki, ExpressRoute, DCI |
| Identity | IAM_ | AD, Entra ID, MFA, PAM |
| Services | SVC_ | DNS, NTP, monitoring, backup |
| Applications | APP_ | Front end, middleware, back end, APIs |
| Data | DATA_ | Databases and storage |

## Visio

Native Visio stencil generation is a later-stage effort. The immediate standard is:
- Draw.io library
- SVG assets
- documented Visio master catalog
